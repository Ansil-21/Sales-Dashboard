# Sales-Dashboard
## This project contains a complete Excel-based data analysis and interactive dashboard created using the sales dataset of Vrinda Store for the year 2022.

The dashboard has been designed to analyze customer purchase patterns, sales performance, order status, and geographical contribution to help improve business decisions and target marketing strategies.
